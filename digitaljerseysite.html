We use basecamp as a reference tool throughout the course. It contains the entire course curriculim as well as a calendar with all the key dates.

We use Treehouse as a tool to learn in our own time - Treehouse has a range of video courses, quizzes and code challenges we use to supplement the sessions.

We use Slack as a tool to communicate with each other. Slack is where we go to ask for help, speak to the mentors or organize meet-ups with other students.
